postcss-gradient-transparency-fix visual tests

This is a page for manually checking the visual output of the postcss-gradient-transparency-fix plugin. At some future point this might get run through automated visual regression tests, but for now it’s just a sanity check.

  • If this page is opened in Safari, it shows the gradients before/after fixing. There should be a huge difference in the gradients in each pair.
  • If this page is opened in any other browser, each test should show two identical gradients. If they’re not identical, the plugin is wrong.
